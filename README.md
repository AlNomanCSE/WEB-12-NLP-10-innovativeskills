# Python Fundamental → Intermediate
### Innovative Skills — NLP-10 Batch

A structured **10-class curriculum** taking you from zero Python knowledge to confident intermediate-level programming.

---

## Curriculum Overview

```
Variables → Control Flow → Loops → Data Structures → Functions → OOP → File Handling → Exceptions → Modules
```

---

## Class-by-Class Breakdown

### Class 1 — Python Basics & Environment Setup
> Getting started with Python and writing your first program.

- Installation — Python + VS Code / PyCharm
- First program: `print()`, comments
- **Variables & Data Types:** `int`, `float`, `str`, `bool`, `NoneType`
- `type()`, `input()`, type casting

---

### Class 2 — Operators & Control Flow
> Making decisions in code using conditions and operators.

- Arithmetic, Comparison, Logical, Assignment operators
- `if`, `elif`, `else`
- Nested conditions
- `match-case` (Python 3.10+)

---

### Class 3 — Loops & Iteration
> Repeating tasks efficiently with loops.

- `for` loop, `while` loop
- `range()`, `break`, `continue`, `pass`
- Nested loops
- Loop with `else`

---

### Class 4 — Data Structures I: List & Tuple
> Storing and managing ordered collections of data.

- **List** — create, index, slice, methods (`append`, `remove`, `sort`, etc.)
- List comprehension
- **Tuple** — immutability, packing/unpacking
- When to use list vs tuple

---

### Class 5 — Data Structures II: Dictionary & Set
> Working with key-value pairs and unique collections.

- **Dictionary** — CRUD operations, methods, nested dict
- Dict comprehension
- **Set** — unique values, set operations (union, intersection, difference)

---

### Class 6 — Functions & Scope
> Writing reusable, clean, and modular code.

- `def`, parameters vs arguments
- Default, keyword, `*args`, `**kwargs`
- Return values
- Scope: local, global, `nonlocal`
- Lambda functions
- Recursion (basic)

---

### Class 7 — Object-Oriented Programming I (OOP Basics)
> Modeling real-world entities using classes and objects.

- Class & Object
- `__init__()`, instance variables, methods
- `self` keyword
- Class variables vs instance variables
- `__str__()`, `__repr__()`

---

### Class 8 — Object-Oriented Programming II (OOP Advanced)
> Taking OOP to the next level with inheritance and polymorphism.

- **Inheritance** — single, multilevel, multiple
- `super()`
- **Encapsulation** — private/protected attributes (`_`, `__`)
- **Polymorphism** — method overriding, duck typing
- **Abstraction** — `ABC`, `@abstractmethod`

---

### Class 9 — File Handling & Exception Handling
> Reading/writing files and handling errors gracefully.

- File open/read/write/append — `open()`, `with` statement
- Text file vs Binary file
- `try`, `except`, `else`, `finally`
- Custom exceptions
- `raise`

---

### Class 10 — Modules, Packages & Pythonic Code
> Organizing code into modules and writing idiomatic Python.

- `import`, `from...import`, `as`
- Built-in modules: `os`, `sys`, `math`, `random`, `datetime`
- Creating your own module/package
- `pip`, virtual environment (`venv`)
- List/Dict/Set comprehension review
- `enumerate()`, `zip()`, `map()`, `filter()`
